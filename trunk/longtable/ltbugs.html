  • tools/3744 (2004-11-22): The longtable environment does not respect \abovecaptionskip.

  • tools/3619 (2004-02-18): longtable leaves single rule on page break
    Bad break on initial hline.
  • tools/3512 (2003-03-09): single-page longtable below float clobbers \pagegoal
    Plausible looking oatch in OR, confirmed to fix some cases on stackexchange.
  • tools/3511 (2003-03-09): longtable does not respect maxdepth
    Similar note to 3510 as regards glue, but need to check maxdepth.
  • tools/3510 (2003-03-09): longtable baselines do not line up at page bottoms
    Not a bug, latex tables designed not to have a strech glue, but taken as a possible feature request for an option to avoid the glue being added.
  • tools/3509 (2003-03-09): longtable starting at page boundary adds extra glue
    I'm not sure this is a bug, adding the forced page break pads with fil glue, so it only interferes with glue on the page if there is fil glue on the page as in the supplied test, not sure \vfil is really supported in that way, fill glue would have won. Adding test file in any case.
  • tools/3488 (2002-12-17): Empty footnote after page ending with longtable
    Analysis and fix by Chung-chieh Shan looks right.
  • tools/3485 (2002-12-10): longtable: more space below than above
    Dup of tools/3180 but with analysis and code fix from Uwe L"uck
  • tools/3479 (2002-11-23): Last page of longtable does not stretch
    Fixed if you comment out vbox.
  • tools/3450 (2002-08-07): Problem with nearly empty pages and longtable.sty
    Sounds like the other ones with a problem with the initial page break, no test file supplied.
  • tools/3407 (2002-02-04): Longtable after \section overrules \nopagebreak
    LT can leave a section head at end of page. No testcase in PR so I made one here.
  • tools/3402 (2002-01-21): tabularx cannot be used inside longtable
    really an ltxtable bug so not really tools, but it's pretty broken here, and ltxtable is mine and possibly should be incorporated into v5 so adding this.
  • tools/3396 (2001-12-11): Bad page break forced by longtable
    Analysis and suggested fix from Lars in the PR.
  • tools/3387 (2001-11-18): captions at left or right aligned longtables
    this is fixed with Axel's ltcaption package, but should probably be fixed here.
  • tools/3180 (2000-03-05): longtable leaves spurious blank line at end
    Analysis from Uwe Lueck in the PR seems about right.
